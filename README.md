# 打砖块游戏项目文档

## 📜 项目概览
- **名称**：打砖块（Arkanoid）
- **类型**：2D 街机类游戏
- **框架**：Pygame Zero (Python 3.7+)
- **目标平台**：Windows

## 🎮 核心功能
### 基础玩法
- **弹球物理**
  支持速度调整、挡板反弹角度计算
- **砖块破坏**
  多种耐久度砖块（命中1次/3次可击碎）
  ```python
  bricks = [
      {"pos": (x, y), "color": color, "durability": 3}
  ]
  ```
- **计分系统**
  击碎砖块得分（+100/块），实时更新最高分记录

### 玩家系统
- **生命值机制**
  可配置初始生命数（默认3条）
- **失败判定**
  球触底后生命值减少，生命耗尽进入游戏结束状态

## 🛠 技术实现
### 架构设计
```plaintext
主循环结构
└── update()   # 60FPS 逻辑更新
    ├── 碰撞检测
    ├── 球运动计算
    └── 状态切换
└── draw()     # 绘制界面
    ├── 砖块渲染
    ├── 文字绘制
    └── 特效显示
```

### 关键技术点
| 模块           | 实现方案                   | 关键代码                       |
|----------------|--------------------------|------------------------------|
| **球反弹逻辑**  | 矢量速度 + 碰撞方向判断    | `ball_speed_x *= -1`          |
| **挡板控制**    | 键盘事件监听             | `keyboard.left/right`        |
